# Royal Note Catalogue — UI Kit

## Overview
Interactive hi-fi prototype of the Royal Note public wholesale catalogue website.

## Screens
- **Catalogue** — product grid with filter bar, search, hero banner
- **Brands** — fragrance house listing page
- **Product Detail** — slide-over panel with variants, MOQ, and CTAs
- **Quote Request** — modal form with confirmation state

## Components
| File | Description |
|---|---|
| `Header.jsx` | Top nav — logo, search, Staff Login, Request Quote |
| `FilterBar.jsx` | Category filter tags + sort dropdown |
| `ProductCard.jsx` | Product card with hover state and badge |
| `ProductDetail.jsx` | Slide-over detail panel |
| `index.html` | Full interactive prototype |

## Design Notes
- Design width: 1280px
- All type uses Playfair Display (serif) for display/product names, Inter for UI
- Tweaks panel: toggle accent color, grid columns, hero visibility
- Product data is synthetic but representative of Royal Note's catalogue structure
